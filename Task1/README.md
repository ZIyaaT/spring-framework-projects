## Description
This is a simple Spring Boot application developed as part of Task 1.
The application demonstrates how to create a Spring Boot project,
define a controller, and handle an HTTP GET request.

When the application is accessed at the root URL (`/`),
it returns a plain text response.

## Technologies Used
- Java
- Spring Boot
- Spring Web
- Thymeleaf
- Lombok
- Maven

## Use Cases

### Use Case 1: Access Root Endpoint
**Actor:** User  
**Description:**  
The user opens a web browser and navigates to `http://localhost:8080/`.

**Expected Result:**  
The application displays the text **HELLO** in the browser.
